## Run Locally

Clone the project

```bash
  git clone https://github.com/Fintrust-Wallet/main-project.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  npm run dev
```
